- **CVR(UU)**: 転換率。閲覧から購入への効率
- **受注点数・受注金額**: 実際の販売実績
- **カート投入数**: 購入意向の強さ

## 活用方針

### 1. リピート発注スコアの精度向上

現在のリピートスコア（粗利率40% + 販売数25% + 季節安定性20% + 併売率15%）に、以下の指標を追加・調整可能:

- **お気に入り率スコア**: お気に入り率が高い品番は今後の売上ポテンシャルが高い → スコア加点
- **CVRスコア**: CVR(UU)が高い品番は効率的に売れている → 在庫持ち月数を増やす根拠
- **UUトレンド**: 日別UUの推移から、アクセスが増加傾向か減少傾向かを判定 → 需要予測の補正

### 2. 広告出稿の指標

- **UUが高くCVRが低い品番**: 閲覧されているが購入に至らない → 価格調整 or 広告で後押し
- **お気に入り率が高くUUが低い品番**: 見た人の評価は高いが露出不足 → 広告で認知拡大
- **カート投入数が多く受注点数が少ない品番**: カゴ落ち率が高い → クーポン施策の対象

### 3. SALE値付けの基準

- **お気に入り率が高い品番**: セール値引き幅を抑えてもよい（需要が強い）
- **CVRが低くUUが高い品番**: 値引きで転換率を上げる効果が期待できる
- **在庫回転が遅くお気に入り率も低い品番**: 大幅値引きの候補

### 4. お気に入り率分析（最重要）

- **MEN vs WOMEN**: 性別ごとのお気に入り率の傾向差を把握
- **カテゴリ別**: どのカテゴリでお気に入り率が高いか
- **品番ランキング**: お気に入り率TOP品番の特徴分析
- **日別トレンド**: お気に入り率の日次変動から新商品の反応を早期検知

## 注意事項

- 現在のデータは **2026年3月の1ヶ月分のみ**。月次で更新される想定
- 販売価格が `#DIV/0!` となっている行がある（受注点数0の場合）
- お気に入り率も受注0の場合は0となる
- 日別データのため、月間集計時は合算が必要（UU・PVは単純合算不可、ユニーク性に注意）
