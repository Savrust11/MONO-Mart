## MDスコア算出ロジック

```
MDスコア = 需要スコア(40%) + 粗利スコア(25%) + 効率スコア(20%) + 成長スコア(15%)

需要スコア: 買い回り件数の対数正規化 × 100
粗利スコア: max(0, 粗利率) / 60 × 100（上限100）
効率スコア: 型あたり売上の対数正規化 × 100
成長スコア: プロパー比率ベース（高いほど良い）
```

## アクション分類

| MDスコア | 内部データあり | アクション |
|---|---|---|
| ≥50 | あり | 強化推奨 |
| 25-49 | あり | 維持 |
| <25 | あり | 見直し |
| any | なし（未展開） | 参入検討 or 要調査 |
