# ZOZO リピート発注シミュレーションスキル

ZOZOの月別実績データから、品番ごとの適正なリピート推奨数量（追加発注数）を算出する。
2パターン（ペナルティなし/強化版）で比較し、品番ごとの発注判断を支援する。

## ワークフロー

1. **データ準備**: 対象品番の月別販売実績、現在庫、粗利率、スコア要素、直近GM変化をスプレッドシート等から取得
2. **シミュレーション実行**: `scripts/calc_repeat_simulation.py` を実行
3. **結果の分析・報告**: 2パターンの推奨数量を比較し、スプレッドシートに反映して報告

## シミュレーション実行

```bash
python scripts/calc_repeat_simulation.py <入力JSON> <出力JSON> [--coefficients <係数JSON>]
```

- 入力JSON: `list[dict]` 形式。各品番に以下のキーを含む:
  - `item_id`, `category`（ZOZO子カテゴリ名）, `current_inv`（現在庫）
  - `monthly_sales`（12要素配列, 4月始まり）
  - `gm_score`, `sales_score`, `season_score`, `cross_score`（各スコア要素, 0-100）
  - `gm_rate`（粗利率%）, `gm_change`（直近2ヶ月のGM変化pp）
- 係数JSON: 省略時は `references/monthly_coefficients.json` を使用

## 予測方式

品番の販売実績期間と月間販売数に応じて4パターンを切り替える。

| 条件 | 方式 | 説明 |
|---|---|---|
| 月販1,000着以下 & 実績3ヶ月以下 | **フラット** | 季節係数を適用せず直近月平均をそのまま使用（新商品の過大予測防止） |
| 月販1,000着以下 & 実績4〜5ヶ月 | **季節係数（単純平均）** | ベース販売力の単純平均を使用 |
| 月販1,000着以上 & 実績5ヶ月以下 | **季節係数（中央値）** | ベース販売力の中央値を使用 |
| 実績6ヶ月以上 | **季節係数（直近重み）** | 直近3ヶ月60%+全体40%の重み付き平均。中央値の2倍を上限キャップ |

ベース販売力 = 各月の実績販売数 / その月の季節係数（係数0.05以下の月は除外）

## スコアと在庫持ち月数

### スコア重み（100点満点）

| 要素 | 重み |
|---|---|
| 粗利率スコア | **40%** |
| 販売数スコア | **25%** |
| 季節安定性スコア | **20%** |
| 併売率スコア | **15%** |

### 在庫持ち月数

| スコア | 在庫持ち月数 | 判定 |
|---|---|---|
| 75点以上 | 5.0ヶ月 | 最優先リピート |
| 60〜74点 | 4.0ヶ月 | 優先リピート |
| 40〜59点 | 3.0ヶ月 | 標準リピート |
| 25〜39点 | 2.5ヶ月 | 慎重リピート |
| 25点未満 | 2.0ヶ月 | 最小限リピート |

## 需要算出（ピーク月選択方式）

8月〜3月の8ヶ月間から、**季節係数の高い順にN月分を選択**して需要合計を算出する。
単純に先頭N月を合計するのではなく、そのカテゴリの売れるシーズンを優先的に選択する。

端数月（例: 2.5ヶ月）は、上位2月の全量 + 3番目の月の50%。

## ペナルティロジック（パターン②）

基本スコアからペナルティを減点する（合計上限30pt）。

| ペナルティ | 発動条件 | 計算式 |
|---|---|---|
| GM低下ペナルティ | 直近2ヶ月のGMが **-2pp以上** 低下 | 低下幅 × 0.8 |
| 粗利率絶対値ペナルティ | 粗利率が **30%未満** | (30 - 粗利率) × 1.0 |

## 推奨数量の算出式

```
リピート推奨数量 = max(0, ピーク月選択による需要合計 − 7月末残在庫)
```

7月末残在庫 = 現在庫から4〜7月の予測販売数を順次差し引いて算出（下限0）。

## 季節係数の更新

カテゴリ別月別係数は `references/monthly_coefficients.json` に格納。
52週売上データから算出した精密な係数を使用している。
新カテゴリ追加や係数更新時はこのJSONファイルを差し替えること。

対応カテゴリ: Tシャツ/カットソー, シャツ/ブラウス, セットアップ, スラックス, デニムパンツ, テーラードジャケット, スウェット, パーカー, ブルゾン, デニムジャケット, モッズコート, トレンチコート
