# データスキーマ定義

## 買い回りデータ（ブランド別シート）

ファイル例：`買い回りデータ2026年2月.xlsx`

- シート名 = ブランド名（例：MONO-MART, CLEL, ADRER, ...）
- 1シート1ブランドの構成

| カラム名 | 型 | 説明 |
|---------|-----|------|
| 会員性別 | string | 男性 / 女性 |
| 年齢区分 | string | 10代, 20代前半, ... |
| ショップ名 | string | 買い回り先の他社ショップ名 |
| 商品カテゴリ | string | 買い回り先での購入カテゴリ |
| 販売価格 | int | 買い回り先での販売価格 |
| 売上点数 | int | 買い回り先での購入点数 |
| 売上金額 | int | 買い回り先での購入金額 |

**買い回りの定義**：同月内に自社ブランドで購入したユーザーが、同月内に他社ショップでも購入した実績。

## ショップ指標データ（月次）

ファイル例：`26年2月.xlsx`, `26年1月.xlsx`

- シート名：Sheet1（固定）
- **ヘッダーが2行目にある**（1行目はタイトル行）→ `header=None` で読み込むこと
- 行数：約1,300行（ZOZOTOWNの全ショップ）

| カラム名 | 型 | 説明 |
|---------|-----|------|
| ショップ名 | string | ZOZOTOWNのショップ名 |
| 受注売上 | float | 月次受注売上（円） |
| 受注数 | int | 月次受注件数 |
| ゲスト購入者注文数 | int | ゲスト購入者の注文数 |
| 販売枚数 | int | 月次販売枚数 |
| 新規再入荷品番数 | int | 新規・再入荷の品番数 |
| 購入者数 | int | 月次ユニーク購入者数（**相性スコアの母数**） |
| 購入者男性比率 | float | 0.0〜1.0 |
| 購入者女性比率 | float | 0.0〜1.0 |
| 購入者平均年齢 | float | ゲスト購入者を含まない |
| 企業名 | string | 運営企業名 |
| 昨年同月売上 | float | 昨年同月の受注売上（円） |
| 昨年同月販売枚数 | int | 昨年同月の販売枚数 |
| 昨年同月1枚単価 | float | 昨年同月の1枚単価（円） |
| 売上昨対比 | float | 売上の昨年同月比（1.0 = 100%） |
| 販売枚数昨対比 | float | 販売枚数の昨年同月比 |
| 1枚単価昨対比 | float | 1枚単価の昨年同月比 |
| 1枚単価 | float | 当月の1枚単価（円） |
| 1件平均 | float | 1件あたり平均購入金額（円） |
| 客単価 | float | 購入者1人あたり購入金額（円） |
| セット率 | float | 1件あたり平均購入点数 |
| 今年度累計売上 | float | 今年度（4月〜当月）累計売上（円） |
| 前年度累計売上 | float | 前年度同期間累計売上（円） |
| 年度累計売上昨対比 | float | 年度累計売上の昨対比 |
| 今年度累計販売枚数 | int | 今年度累計販売枚数 |
| 前年度累計販売枚数 | int | 前年度同期間累計販売枚数 |
| 年度累計販売枚数昨対比 | float | 年度累計販売枚数の昨対比 |

## 相性スコアの算出式

```
相性スコア(%) = 買い回り件数 / ショップ購入者数 × 100
```

- **買い回り件数**：買い回りデータの `ショップ名` でグループ化した `count()`
- **ショップ購入者数**：ショップ指標データの `購入者数` カラム

## データ結合のキー

`ショップ名`（文字列完全一致）で結合する。
マッチしないショップ（購入者数がNaN）はスコア算出から除外される。
